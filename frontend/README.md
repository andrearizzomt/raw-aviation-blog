# Next.js frontend (`frontend/`)

App Router frontend for RAW Aviation Blog. **Project docs:** [../README.md](../README.md) · **Railway:** [../RAILWAY_DEPLOYMENT_PATH.md](../RAILWAY_DEPLOYMENT_PATH.md)

## Commands

| Command | Purpose |
|---------|---------|
| `npm run dev` | Local dev (`http://localhost:3000`, Turbopack) |
| `npm run build` | Production build |
| `npm run start` | Production server (after `build`) |
| `npm run lint` | ESLint |

Set `NEXT_PUBLIC_STRAPI_API_URL` in `.env.local` (e.g. `http://localhost:1337` for local Strapi).

Next.js documentation: [nextjs.org/docs](https://nextjs.org/docs).
