# Strapi CMS (`cms/`)

Headless CMS for RAW Aviation Blog. **Project docs:** [../README.md](../README.md) · **Railway:** [../RAILWAY_DEPLOYMENT_PATH.md](../RAILWAY_DEPLOYMENT_PATH.md) · **Env / prod:** [../DEPLOYMENT.md](../DEPLOYMENT.md)

## Commands

| Command | Purpose |
|---------|---------|
| `npm run develop` | Local dev with auto-reload (`http://localhost:1337`) |
| `npm run build` | Production admin build |
| `npm run start` | Production server (after `build`) |
| `npm run seed` | Sample content (requires `SEED_*` in `.env`; see [../CLAUDE.md](../CLAUDE.md)) |

Official Strapi CLI reference: [Strapi docs — CLI](https://docs.strapi.io/dev-docs/cli).
